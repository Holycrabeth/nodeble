# Funding Strategy

---

## Approach: 100% Self-Funded (Bootstrapped)

No external funding. No investors. No dilution.

### Why This Works

| Factor | Detail |
|--------|--------|
| **Phase 1 cost** | $0 — helping friends for free, code already exists |
| **Phase 2 cost** | Minimal — founder's time + existing infrastructure |
| **Infrastructure** | Already owned: Mac Mini, GPU server, NAS |
| **Cloud costs** | Friends pay their own Vultr VPS (verify current pricing at vultr.com — free tier available) |
| **Software tools** | Claude Code subscription (already paying) |
| **Legal** | ~SGD 5-15K for regulatory legal opinion (one-time, before commercial launch) |
| **Runway** | Founder is retired — no salary pressure, no burn rate |

### Total Estimated Cost to Commercial Launch

| Item | Cost | When |
|------|------|------|
| Legal opinion (MAS/regulatory) | SGD 5,000-15,000 | Before Phase 3 |
| Domain + hosting for product site | ~$20/month | Phase 2 |
| Apple Developer account (if Mac app) | $99/year | Phase 2 |
| Marketing (if any) | ~$500-1,000 | Phase 3 |
| **Total to first paying customer** | **~SGD 7,000-17,000** | |

This is pocket change for a retired founder with $300K+ in trading accounts.

### Revenue Reinvestment

Once revenue starts:
- First $1K MRR → covers all operating costs
- First $5K MRR → consider hiring freelance help
- First $10K MRR → consider full-time hire or infrastructure scaling

### When External Funding Might Make Sense

Only if:
- Product-market fit is proven (50+ paying users)
- Growth is constrained by capital (e.g., need to build mobile app, hire team)
- A strategic investor (e.g., Tiger Brokers themselves) offers partnership
- **NOT for Phase 1-3.** Bootstrapping is the right model until the product clearly works.
